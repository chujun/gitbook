# 观点
冥想是一项技能，又是一种体验

要想理解冥想，你需要取践行冥想-->正如别人天花乱坠地描述蹦极多么刺激，多么激动人心，你都无法想象自己从悬崖那下一跃而下的体验。


# 现实运用
* 医院界冥想被用来治疗与压力相关的症状，包活长期焦虑，抑郁，愤怒，失眠等等
* 美国海军曾采用冥想使士兵在前线更专注，更有效率
* 科学研究发现冥想过程中大脑结构产生变化，成为"神经可塑性"
