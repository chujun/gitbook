# 毛病
## 
* 摄入坚果类可以改善成年人血脂
* 增加饮水量降低肾脏和泌尿系统感染的风险

## 胃
* 增加蔬菜摄入总量与胃癌发病风险无关,但葱类蔬菜和十字花蔬菜对预防胃癌有保护作用
* 增加水果摄入可降低胃癌的发病风险
* 每天增加6倍绿茶可降低21%胃癌发病风险(适量喝点)
