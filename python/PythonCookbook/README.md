# 关于本书
本地demo都是基于python3.3上运行并通过测试的。不保证对python2的兼容性
聚焦python3的好的最佳编程方式
## 本书适用读者
有一定Python基础，想加深对Python语言理解以及学习编程最佳实践方法的人
## 本书不适用对象
* 首次学习Python语言的初学者
* 不是快速参考手册(例如快速查询特定模块中的某个函数)
